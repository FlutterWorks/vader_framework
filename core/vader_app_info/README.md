# vader_app_info

Give you info about package and device

## Installation

Add the following dependency to your `pubspec.yaml` file:

```yaml
dependencies:
  vader_app_info: ^0.1.0
```

Then run:

```sh
flutter pub get
```

## Usage:

```
import 'package:vader_app_info/vader_app_info.dart';
```
