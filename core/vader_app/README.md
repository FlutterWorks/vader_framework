# vader_app

Vader App Framework for simplify development of Flutter apps.

## Installation

Add the following dependency to your `pubspec.yaml` file:

```yaml
dependencies:
  vader_app: ^0.2.0
```

Then run:

```sh
flutter pub get
```

## Usage:

```
import 'package:vader_app/vader_app.dart';
```
