# vader_core

Core package of Vader Framework.

## Installation

Add the following dependency to your `pubspec.yaml` file:

```yaml
dependencies:
  vader: ^0.4.0
```

Then run:

```sh
flutter pub get
```

## Usage:

```
import 'package:vader_core/vader_core.dart';
```
